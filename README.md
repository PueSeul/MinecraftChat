# Java Chat

Java Chat은 안드로이드에서 Minecraft Java 서버의 채팅과 접속자 목록을 확인할 수 있는 앱입니다. 서버 플러그인 없이 접속하며, Java 1.8.9부터 26.2까지의 서버 버전 선택, 채팅과 명령어 전송, 명령어 자동완성, 접속자 스킨 얼굴 표시, 여러 서버 저장 및 상태 확인을 지원합니다.

## 설치 방법

1. [Java Chat v1.2 APK 다운로드](https://github.com/PueSeul/JavaChat-Android/releases/download/v1.2/JavaChat-v1.2.apk)를 눌러 APK를 받습니다.
2. 다운로드한 `JavaChat-v1.2.apk`를 실행합니다.
3. Android가 요청하면 브라우저 또는 파일 앱의 **이 출처의 앱 설치 허용**을 켜고 설치를 완료합니다.

ADB로 설치하려면 다음 명령을 사용할 수 있습니다.

```powershell
adb install -r JavaChat-v1.2.apk
```

## 사용 방법

1. 첫 화면에서 **온라인** 또는 **오프라인** 접속 방식을 선택합니다.
2. 온라인 접속은 **Microsoft 로그인**을 눌러 브라우저에서 로그인을 완료합니다.
3. 오프라인 접속은 사용할 닉네임을 입력해 저장합니다. 오프라인 모드는 `online-mode=false` 서버에서 사용할 수 있습니다.
4. **서버 추가**를 눌러 서버 이름, 주소, 포트와 Minecraft 버전을 저장합니다.
5. 저장한 서버의 **접속**을 누르면 채팅과 접속자 목록을 확인할 수 있습니다.
6. 채팅 입력란에 메시지나 `/` 명령어를 입력해 전송합니다. 명령어를 입력하면 서버가 제공하는 자동완성 항목이 표시됩니다.
7. 연결을 끝내려면 화면의 **종료**, 뒤로가기 또는 알림의 **서버 나가기**를 사용합니다.
8. 첫 화면의 **업데이트 확인**을 누르면 GitHub의 최신 정식 버전을 확인할 수 있습니다. 새 버전이 있으면 **업데이트**를 눌러 다운로드하고 Android 설치 화면에서 설치를 완료합니다.

# Java Chat v1.0

Minecraft Java 서버의 채팅과 접속자 목록을 안드로이드에서 확인하는 첫 정식 릴리스입니다.

## 주요 기능

- Java 1.8.9–26.2의 65개 릴리스 선택
- Microsoft 계정 및 오프라인 닉네임 접속
- 채팅, 명령어 전송과 서버 기반 자동완성
- 접속자 목록, 지연 시간과 스킨 머리 이미지
- 여러 서버 저장과 온라인 상태 확인
- GitHub 최신 정식 릴리스 확인, APK 자동 다운로드·검증과 업데이트 설치 화면 연결
- 알림의 서버 나가기 버튼과 안전한 연결 종료

## 검증

- 전체 36개 자동 테스트 중 실행 가능한 35개 통과(로컬 다중 버전 서버 테스트 1개는 서버 미실행으로 건너뜀)
- 공식 서버 1.8.9, 1.12.2, 1.16.5, 1.19.4, 1.21.11 통합 테스트
- Android Lint 오류 0개
- Samsung SM-S908N에서 설치 및 ViaVersion 변환기 초기화 확인
- APK Signature Scheme v2 서명 확인

GitHub에서 직접 설치하거나 앱 안에서 업데이트할 때 Android가 이 출처의 앱 설치 허용과 최종 설치 확인을 요청할 수 있습니다.
